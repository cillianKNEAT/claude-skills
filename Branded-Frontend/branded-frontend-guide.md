# Branded Frontend Skill — Quick Reference

Your `branded-frontend` skill now includes design refinement commands from **Impeccable**. Everything works through natural language — just describe what you need.

---

## Creating New Interfaces

Ask Claude to build any frontend component, page, or app. The skill automatically enforces your Kneat brand tokens (Muli + Open Sans, Midnight Navy / Java / Polo Blue palette, 8/16/32/40/80px spacing, responsive breakpoints).

> *"Build me a pricing page for our product"*
> *"Create a React dashboard component showing user metrics"*
> *"Design a landing page for our new feature launch"*

---

## Refinement Commands

Use these keywords naturally in conversation to trigger specific design workflows. Paste or reference your code/component alongside the request.

| Keyword | What it does | Example prompt |
|---|---|---|
| **audit** | Find issues without fixing them — generates a severity-rated report | *"Audit this component for accessibility and brand compliance"* |
| **critique** | Holistic design review like a design director | *"Critique this layout — what's working and what's not?"* |
| **polish** | Final quality pass — alignment, states, consistency | *"Polish this page, it's almost ready to ship"* |
| **animate** | Add purposeful motion and micro-interactions | *"Animate the card grid with staggered entrance reveals"* |
| **bolder** | Amplify underwhelming designs (within brand) | *"Make this hero section bolder"* |
| **quieter** | Tone down overly intense designs | *"This dashboard feels too heavy — make it quieter"* |
| **distill** | Strip to essence, remove complexity | *"Distill this settings page — it's overwhelming"* |
| **colorize** | Add strategic color from the brand palette | *"Colorize this — it's too monochromatic"* |
| **clarify** | Improve UX copy, labels, error messages | *"Clarify the error messages in this form"* |
| **adapt** | Make responsive across breakpoints | *"Adapt this for mobile and tablet"* |
| **normalize** | Align with brand design tokens | *"Normalize this — it's using off-brand colors and fonts"* |
| **extract** | Pull out reusable components and tokens | *"Extract the repeated card pattern into a shared component"* |
| **optimize** | Improve performance (loading, rendering, bundle) | *"Optimize this page — it's loading slowly"* |
| **harden** | Handle edge cases, i18n, overflow, errors | *"Harden this form for long names and network failures"* |
| **onboard** | Design onboarding flows and empty states | *"Design an onboard experience for first-time users"* |
| **delight** | Add subtle moments of personality | *"Add some delight to the success state after saving"* |

---

## Tips

- **Combine commands**: *"Audit this page, then polish the issues you find"*
- **Be specific**: *"Make the hero section bolder but keep the sidebar quieter"*
- **Provide context**: Paste your code, share a screenshot, or describe what you've built — the more context, the better the output
- **Brand always wins**: The skill will never suggest off-brand fonts, colors, or spacing — all refinements stay within your Kneat design system
